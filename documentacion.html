Introduction

Welcome to the Translator Power Bot documentation! We will give you an explanation of how to use the bot and the commands!

The prefix of the bot is ".". At this moment, you can not change the prefix, but we are working on that.

This bot helps you to comunicate between people of all around the world!

Translate commands

In this section we are going to talk about the translate commands in the bot

.lang

The .lang command displays information about the current language on the server

.set_lang

With the .set_lang command you can set the language of the server

Example, configure the server to spanish

.set_lang es

.set_channel

This command it services to translate all messages of select channel the language select in him

.set_deletech

This command is used to eliminate the automatic translation of the channel selected with 'set_channel'

.info_bot

This command it services to show information of bot

.help

This command it services to show link for documentation bot

.ts

This command is used to translate words from any language to the language of the server
